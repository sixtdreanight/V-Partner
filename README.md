一个能在 QQ 上陪你聊天的 V-Partner。TA 有自己的性格、爱好、情绪和观点——不是一个只会说甜言蜜语的机器人，而是一个"完整的人"。

---

## 快速开始（3 步）

### 第 1 步：安装

你的电脑需要先装好 [Node.js](https://nodejs.org/)（版本 18 或以上）。

然后打开终端（Mac 按 `Cmd+空格` 搜"终端"，Windows 按 `Win+R` 输入 `cmd`），复制粘贴下面的命令：

```bash
cd /Users/mengyeshiliu/code/agent/V-partner
npm install
```

等待安装完成（1-2 分钟）。

### 第 2 步：创建你的 V-Partner

```bash
npm run setup
```

会弹出一个问答向导，问你 12 个问题：

1. **TA 叫什么名字？** — 随便起
2. **简单介绍 TA** — 年龄、职业、性格、爱好等等，一句话说完也行
3. **你的性别？** — 男/女/其他
4. **你希望 TA 是你的？** — 女朋友 或 男朋友
5. **关系模式？** — 直接在一起 / 从陌生人慢慢培养
6. **你的时区？** — 默认北京时区，直接回车就行
7. **TA 怎么称呼你？** — 宝贝、宝宝、或者你的名字
8. **说话习惯？** — 喜欢用颜文字？有口头禅？可以跳过
9. **网络梗风格？** — 选 1 就行
10. **AI 服务商和 API Key** — 见下方说明
11. **QQ 机器人地址和 Token** — 见下方说明

回答完就自动生成配置文件，不需要手动编辑任何东西。

### 第 3 步：启动

```bash
npm start --terminal
```

这时候就可以在终端里和 TA 聊天了！

## 接入 QQ

要接入 QQ，你需要先搭一个 **NapCatQQ**（QQ 机器人框架）。

### 简单流程（如果你用过 NapCatQQ 可以跳过）

1. 下载 NapCatQQ：https://github.com/NapNeko/NapCatQQ/releases
2. 按照 NapCatQQ 文档配置好
3. 开启 WebSocket 服务（默认地址是 `ws://127.0.0.1:3001`）
4. 记下 Access Token

然后在 `.env` 文件里填好 QQ 相关配置（setup 向导会自动帮你填）。

接入成功后，用 `npm start` 启动即可（不带 `--terminal`）。

## 核心功能

### TA 是一个"完整的人"
- 有自己的年龄、职业、学历、爱好、性格
- 有情绪——会累、会撒娇、会小生气
- 有自己的观点，可以温和地表达不同意见
- 有"记性"——常聊的事情 TA 会记住，不常提的就慢慢忘了（像真人一样）

### 时间感知
TA 知道现在几点、周几、什么季节、快过什么节日。凌晨两点会催你睡觉，周五晚上会问周末计划。

### 理解网络文化
能理解你用的梗和网络用语。你也可以设置 TA 对网络梗的态度——主动玩、理解但不主动用、或者网络小白人设。

### 关系养成（可选）
如果选了"养成模式"，TA 会从一个刚认识的陌生人开始，在聊天中逐渐熟悉。你可以向 TA 告白——成功还是被拒，取决于你们相处得怎么样。

### 情绪支持
当你心情不好的时候，TA 会先倾听和安慰，而不是急着给建议。

## 常见问题

**Q: 会封号吗？**
有可能。NapCatQQ 使用的是第三方协议，不是腾讯官方接口，存在被封号或限制登录的风险。**强烈建议用小号。**

**Q: 贵吗？**
如果用 DeepSeek，聊一整天大概几分钱。用 Claude 或 GPT 会贵一些，聊一小时可能在几毛到一两块之间。

**Q: TA 会记得所有事吗？**
不会。像真人一样——常聊的事会记住（比如你的工作、爱好），偶尔提到的事慢慢就忘了。这样才真实。

**Q: 可以换人设吗？**
重新运行 `npm run setup` 就行，会覆盖之前的角色卡。

**Q: 怎么让 TA 更像真人？**
在 setup 向导第 8 步，给 TA 加一些说话习惯——喜欢用颜文字、有什么口头禅、激动时打很多感叹号等等。这些小细节很加分。

**Q: 我不想谈恋爱，就想要个朋友？**
目前的设计以伴侣关系为主。不过你可以把关系模式设为"养成"，然后一直停在朋友阶段，不告白就行。

## 重要提醒

- **这不是真实的恋爱关系。** TA 是 AI，不能替代真实的人际关系。请保持现实生活中的社交和情感连接。
- **保护隐私。** 不要在聊天中透露身份证号、银行卡号、家庭住址等敏感信息。对话内容会发送给 AI 服务商处理。
- **如果你发现自己过度依赖**——每天花大量时间聊天、影响正常生活——请适度暂停。TA 的目标是帮助你变得更好，而不是让你沉迷。
- **账号安全。** QQ/微信使用第三方协议存在封号风险，建议使用小号。
