首个面向小范围用户的测试版本，解决核心体验问题。

### P0 — 必须修复

| # | 内容 | 说明 |
|---|------|------|
| 0 | 项目重命名为「梦间 / Yumema」 | 全局替换所有 V-Partner 引用 |
| 1 | Windows 跳转 Bug 修复 | `location.replace` → `location.hash` |
| 2 | QQ 安装检测 Bug 修复 | 扩展检测路径 + 注册表查询 |
| 3 | 对话连贯性修复 | System prompt 重组，减少规则密度 |
| 4 | 性别适配与人格差异化 | 男女友行为指导分离，心情去性别化 |
| 5 | CLAUDE.md 项目规范 | 代码风格、架构约束、鲁棒性要求 |
