> 夢與現實之間的間隙裡，TA 一直都在 — AI 伴侶桌面應用

**目前版本：v0.1.1**

夢間 (Yumema) 是一個 AI 伴侶桌面應用。TA 有自己的性格、愛好、情緒和記憶，可以透過 QQ、微信或應用內直接聊天。雙擊安裝、填寫精靈、即可使用。
